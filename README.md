# 🏎️ Super Corrida Pro

Um simulador de corrida arcade desenvolvido com **HTML5**, **CSS3** e **JavaScript** puro (Vanilla JS). O projeto foca em mecânicas de movimento fluido e animações de cenário para criar uma experiência imersiva de velocidade.

## 🕹️ Como Jogar
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Pressione a tecla **ESPAÇO** para dar a largada.
4. Use as **SETAS** do teclado para controlar o carro e desviar (em breve obstáculos!).

## 🚀 Funcionalidades
- **Cenário Dinâmico:** Estrada com calçadas zebradas e movimento infinito.
- **Movimentação Suave:** Controle do jogador otimizado com `requestAnimationFrame`.
- **Sistema de Pontuação:** Score baseado na distância percorrida (tempo de sobrevivência).
- **Design Responsivo:** O jogo é centralizado e adaptado para diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura dos elementos do jogo.
* **CSS3:** Estilização, degradês de pista e animações visuais.
* **JavaScript:** Lógica de movimento, detecção de teclas e loop do jogo.
